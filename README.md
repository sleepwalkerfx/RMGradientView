# Installing 

CocoaPods is the preferred way to install this library. Add this command to your `Podfile`:

```
pod 'RMGradientView', :git => 'https://github.com/sleepwalkerfx/RMGradientView.git
```

# Try It


```
pod try RMGradientView
```
